Punishing Gray Raven API

A repository dedicated to storing information on every playable construct within game.
Updated to the current CN patch.

API endpoint: https://pgr-api.vercel.app/api/

  • Get the full list of constructs: https://pgr-api.vercel.app/api/
  • Query the API for a specific construct by construct title:
    https://pgr-api.vercel.app/api/scire

API Calls & Endpoints

HTTP method Endpoint Reponse
GET /api Returns an array of all the construct within this database.
GET /api/:construct Returns an object containing information of the given construct within this database. Must use constructs title. Example: 'Scire' not 'Karenina'.
